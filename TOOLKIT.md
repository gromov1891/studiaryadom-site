# Инструментарий для проекта studio-site

Каталог агентов и скиллов, полезных при разработке премиального сайта студии.
Отобрано из воркспейса «Курс» под этап разработки сайта. Клиент-специфичные
скиллы (Tantal / Villo / Harmonia / KP / юристы / музыка) сюда НЕ включены —
они про другие проекты.

Условные обозначения:
- 🟢 **проектный агент** — создан для studio-site (`.claude/agents/`)
- 🔧 **скилл** — вызывается как `/имя`
- 🤖 **агент-тип** — вызывается через Agent (`subagent_type: "имя"`)

---

## 1. Стратегия и планирование
| Что | Тип | Когда нужен |
|---|---|---|
| `office-hours` | 🔧 | Продумать продукт/структуру сайта до кода: какие секции, какой оффер. |
| `plan-ceo-review` | 🔧 | «Думать крупнее» — поднять амбицию концепции до уровня референса. |
| `plan-eng-review` | 🔧 | Ревью технического плана перед реализацией фичи. |
| `feature-dev` | 🔧 | Управляемая разработка фичи с анализом кодовой базы. |
| `system-designer` | 🤖 | Архитектура (если появятся бэкенд/CMS/формы заявок). |
| `wireframe-creator` | 🤖 | Каркасы экранов и пользовательские потоки до визуала. |

## Минимальный рабочий цикл одной секции
1. `copywriter-ru` → тексты секции.
2. `frontend-designer` → вёрстка и композиция.
3. `animation-engineer` → движение поверх.
4. `node scripts/shot.mjs` → скриншоты 1440/390.
5. `design-reviewer` → список правок по скриншотам → возврат к шагу 2.
6. `seo-auditor` перед публикацией страницы.
