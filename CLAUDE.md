# Премиальный сайт видеопродакшн-студии (Москва)

Самописный премиальный сайт студии видеопродакшна. Рынок — РФ, язык интерфейса и контента — **русский**.
Референс уровня полировки: https://seo-lebedev.ru/ (тёмная тема, крупная типографика, бесконечная 3D-лента кейсов, плавный скролл, прелоадер с процентами, появление по скроллу). **Равняемся на этот уровень, но не копируем дизайн.**

> Бренд и домен заданы в одном месте — `lib/site.ts`. Сейчас там плейсхолдер. Замена — только там.

---

## Стек (жёстко)

- **Next.js 16** (App Router, TypeScript) — *в брифе была 15; `create-next-app@latest` ставит 16, она новее и совместима.*
- **Tailwind CSS v4** — конфиг CSS-first через `@theme` в `app/globals.css` (никакого `tailwind.config.js`).
- **GSAP + ScrollTrigger** — основной движок скролл-анимаций.
- **Lenis** — плавный скролл (smooth scroll).
- **Framer Motion** — **только** для микроинтеракций (hover, мелкие появления). Не для скролл-сцен.
- **Никаких UI-китов**: без shadcn/ui, MUI, Bootstrap, Ant и т.п. Всё своё.

## Важно про Next.js 16

Это новая мажорная версия с breaking changes относительно знаний по умолчанию. При сомнениях — читать `node_modules/next/dist/docs/`. Главное:
- `params` и `searchParams` теперь **только асинхронные** (`await props.params`). На статичных страницах без динамических сегментов не важно.
- Минимум Node.js 20.9, TypeScript 5.1, React 19.2.

⚠️ **Сборка на webpack, НЕ на Turbopack.** Путь проекта содержит кириллицу (`Рабочий стол\Курс`), а Turbopack на этом падает (`panic: not a char boundary` в Rust-коде идентификаторов). Поэтому в `package.json` стоит `next dev --webpack` и `next build --webpack`. Не возвращай Turbopack, пока проект лежит под кириллическим путём (либо перенеси проект в ASCII-путь, тогда можно вернуть Turbopack).

## Принципы анимаций

- Анимируем **только `transform` и `opacity`**. Никогда — `width`/`height`/`top`/`left`/`margin` (reflow → тормоза).
- `prefers-reduced-motion: reduce` — **обязателен**. При нём отключаем параллакс/marquee/SplitText, оставляем мгновенное появление.
- **60 fps — жёсткое требование.** GPU-слой через `transform: translate3d`, `will-change` точечно.
- Любая GSAP-логика в React — внутри `gsap.context()` с чисткой `return () => ctx.revert()` (корректный unmount, нет утечек ScrollTrigger).
- Никакого layout shift (CLS): резервируем размеры под медиа и шрифты.

## Дизайн-принципы

- **Тёмная тема по умолчанию.** Фон глубокий, не чистый чёрный (#0a0a0b).
- **Крупная выразительная типографика** — вариативный гротеск, **кириллица обязательна**. Дисплейный — Unbounded, текстовый — Onest (оба `next/font/google`, сабсет `cyrillic`).
- **Много воздуха** — крупные отступы, дышащая сетка.
- **Никакого «дефолтного Tailwind-вида»**: без стоковых сине-фиолетовых градиентов, без одинаковых скруглений на всём, без центрированного текста везде, без emoji в UI.
- Дизайн-токены (цвет, типошкала, отступы, длительности, easing) — `app/globals.css` (`@theme`) + зеркало для JS в `lib/design-tokens.ts`.

## SEO (рунет)

- Семантический HTML (один `<h1>` на страницу, иерархия `h2/h3`, `<nav>`, `<main>`, `<section>`, `<article>`).
- **schema.org** (JSON-LD): `Organization`, `LocalBusiness`, `Service`, `BreadcrumbList`.
- Мета-теги и OpenGraph — **на русском**.
- `sitemap.xml` (`app/sitemap.ts`), `robots.txt` (`app/robots.ts`).
- Русскоязычные URL — **транслитом** (`/uslugi`, `/keysy`, `/kontakty`), не кириллицей в путях.
- `alt` у всех значимых изображений, на русском.
- Цель — **95+ в Lighthouse по всем категориям**. Веб-виталс: LCP < 2.5s, CLS < 0.1, INP < 200ms.
- Специфика Яндекса: турбо не нужны, критичны микроразметка и быстрая отдача (SSR/статика, лёгкий JS).

## Рабочий цикл (обязательно)

- После **каждого** визуального изменения — скриншот (desktop **1440px** и mobile **390px**), посмотреть, итерировать.
- **Задача не выполнена без визуальной проверки.** «Готово» = я видел скриншот и он на уровне референса.
- Скриншоты: `node scripts/shot.mjs` (Playwright) либо Playwright MCP, когда подключён.
